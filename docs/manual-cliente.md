# Guía del Cliente — LuminaSupport

---

## ¿Qué es LuminaSupport?

LuminaSupport es un sistema que **vigila tu sitio web las 24 horas del día, los 7 días de la semana**. Piensa en él como un guardia de seguridad digital que está siempre pendiente de que tu página funcione correctamente.

Cada pocos minutos, el sistema visita tu sitio web automáticamente para verificar que:
- Tu sitio esté **encendido y funcionando**
- Cargue **rápido** para tus visitantes
- El **certificado de seguridad** (el candadito verde) esté vigente
- Todo el **contenido** se esté mostrando correctamente

Si algo sale mal, el sistema lo detecta de inmediato, crea un registro del problema y notifica al equipo técnico para que lo resuelvan lo antes posible.

## Tu Panel: ¿Qué puedes hacer?

Cuando inicias sesión en LuminaSupport, llegas a tu panel personal. Aquí te explicamos cada sección:

### 1. Mi Panel (Dashboard)

Es la primera pantalla que ves al entrar. Te muestra un resumen rápido de cómo van tus sitios:

- **Mis Sitios**: Cuántos sitios web tienes bajo monitoreo
- **Disponibilidad Promedio**: Un porcentaje que te dice cuánto tiempo han estado funcionando tus sitios. 100% significa que nunca se han caído; 99.9% significa que hubo apenas unos minutos de problema
- **Incidentes Activos**: Si hay algún problema en curso con alguno de tus sitios
- **Gráfico de Disponibilidad**: Una línea visual que te muestra cómo ha sido la disponibilidad en los últimos días
- **Actividad Reciente**: Las últimas cosas que han pasado con tus sitios (actualizaciones, incidentes resueltos, etc.)

### 2. Mis Sitios

Aquí ves la lista de todos tus sitios web que están siendo monitoreados. Para cada sitio puedes ver:

- **Nombre y dirección** del sitio
- **Estado actual**: Si está funcionando (verde), si tiene algún problema (amarillo), o si está caído (rojo)
- **Disponibilidad**: El porcentaje de tiempo que ha estado funcionando
- **Última revisión**: Cuándo fue la última vez que el sistema verificó tu sitio

### 3. Soporte

Si necesitas ayuda o tienes algún problema, aquí puedes:

- **Crear un ticket de soporte**: Describe tu problema o solicitud y el equipo técnico lo recibirá
- **Ver tus tickets anteriores**: Revisa el estado de tus solicitudes pasadas
- **Buscar en tus tickets**: Encuentra rápidamente un ticket específico
- **Ver el estado**: Cada ticket puede estar pendiente, en proceso o resuelto

### 4. Configuración

Aquí puedes personalizar tu cuenta:

- **Perfil**: Tu nombre, empresa, teléfono
- **Notificaciones**: Elige qué tipo de alertas quieres recibir por email
- **Contraseña**: Cambia tu contraseña cuando lo necesites

## ¿Qué pasa cuando mi sitio tiene un problema?

Cuando el sistema detecta un problema con tu sitio, esto es lo que ocurre:

1. **Detección**: El sistema visita tu sitio y nota que algo no funciona
2. **Verificación**: Para estar seguro (y no dar falsas alarmas), espera 30 segundos y vuelve a verificar
3. **Registro**: Si confirma el problema, crea un "incidente" — un registro detallado de qué pasó
4. **Notificación**: Se envía un aviso al equipo técnico (y a ti, si tienes las notificaciones activadas)
5. **Seguimiento**: El sistema sigue revisando tu sitio cada pocos minutos
6. **Resolución**: Cuando tu sitio vuelve a funcionar, el incidente se cierra automáticamente y recibes una notificación de que todo está bien

Todo esto pasa **automáticamente**, sin que tengas que hacer nada.

## ¿Qué significan los colores?

| Color | Significado |
|---|---|
| 🟢 **Verde** | Todo funciona correctamente |
| 🟡 **Amarillo** | Hay una advertencia menor (ej: el sitio está un poco lento, o el certificado de seguridad expirará pronto) |
| 🔴 **Rojo** | Hay un problema serio (ej: el sitio está caído o no responde) |

## ¿Qué es el certificado SSL y por qué importa?

El certificado SSL es lo que hace que tu sitio tenga el **candadito verde** en el navegador y que la dirección empiece con **https://** en vez de http://. Es importante porque:

- **Protege** la información que tus visitantes envían (datos personales, pagos, etc.)
- **Google prefiere** sitios con SSL y los posiciona mejor en sus resultados
- **Los navegadores** muestran una advertencia de "sitio no seguro" si no tienes SSL

LuminaSupport revisa automáticamente cuándo vence tu certificado y te avisa con anticipación para que se renueve a tiempo.

## Preguntas Frecuentes

**¿Con qué frecuencia se revisa mi sitio?**
Cada 3 a 5 minutos, las 24 horas del día.

**¿Me avisarán si mi sitio se cae?**
Sí, el sistema envía una alerta por email inmediatamente. También recibirás un aviso cuando el sitio se recupere.

**¿Qué pasa si hay una falsa alarma?**
El sistema tiene un mecanismo de doble verificación. Si detecta un problema, espera 30 segundos y vuelve a revisar antes de crear una alerta. Esto minimiza las falsas alarmas.

**¿Puedo ver el historial de mi sitio?**
Sí, en "Mis Sitios" puedes ver el historial de disponibilidad y los incidentes pasados.

**¿Cómo creo un ticket de soporte?**
Ve a la sección "Soporte", haz clic en "Nuevo Ticket", describe tu problema o solicitud, y envíalo. El equipo técnico lo recibirá inmediatamente.

**¿Mi información está segura?**
Sí. El sistema usa encriptación y controles de acceso estrictos. Solo puedes ver tus propios sitios y datos — no los de otros clientes.

## Glosario — Términos que puedes encontrar

| Término | ¿Qué significa? |
|---|---|
| **Sitio web** | Tu página en internet (ej: www.tuempresa.cl) |
| **Disponibilidad (Uptime)** | El porcentaje de tiempo que tu sitio ha estado funcionando. 100% = nunca se cayó |
| **Caída (Downtime)** | Cuando tu sitio no funciona y los visitantes no pueden acceder |
| **Incidente** | Un registro de un problema detectado en tu sitio. Se crea automáticamente cuando algo falla |
| **SSL / Certificado de seguridad** | El "candadito verde" que protege tu sitio. Es lo que hace que aparezca https:// |
| **Ticket de soporte** | Una solicitud de ayuda o reporte de problema que envías al equipo técnico |
| **Monitoreo** | La vigilancia constante que el sistema hace sobre tu sitio |
| **Tiempo de respuesta** | Qué tan rápido responde tu sitio cuando alguien lo visita. Menos tiempo = más rápido = mejor experiencia |
| **Plugin** | Un programa adicional instalado en tu sitio web que agrega funciones extras (ej: formulario de contacto, tienda, etc.) |
| **Actualización** | Una nueva versión de un programa de tu sitio (plugin o tema) que incluye mejoras o correcciones de seguridad |
| **WordPress** | El programa con el que está construido tu sitio web. Es como el "motor" de tu página |
| **WooCommerce** | Un agregado de WordPress que convierte tu sitio en una tienda online |
| **Dashboard / Panel** | La pantalla principal donde ves el resumen de toda tu información |
| **Notificación** | Un aviso que recibes (generalmente por email) cuando algo importante pasa con tu sitio |
| **Hosting / Servidor** | El computador donde "vive" tu sitio web. Es el que lo mantiene encendido y disponible para los visitantes |
| **DNS** | El sistema que traduce el nombre de tu sitio (ej: tuempresa.cl) a la dirección del servidor donde está alojado. Es como la guía telefónica de internet |
| **Caché** | Una copia temporal de tu sitio guardada más cerca del visitante para que cargue más rápido |
| **WebP** | Un formato de imagen más moderno y liviano que los tradicionales (PNG, JPG). Hace que tu sitio cargue más rápido |
| **SEO** | Las técnicas que ayudan a que tu sitio aparezca más arriba en Google cuando alguien busca algo relacionado con tu negocio |
| **Cloudflare** | Un servicio que protege y acelera tu sitio web, funcionando como un escudo entre los visitantes y tu servidor |
| **Vercel** | Una plataforma donde pueden estar alojados sitios web modernos. Es como un hosting especializado en sitios rápidos |
| **FTP** | Una forma de subir archivos a tu servidor. Los técnicos lo usan para modificar archivos de tu sitio |
